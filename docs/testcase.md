## Test cases

A test case json file contains an array of test object that contains the necessary to run tests

```json
[
    {
        "testId": "x1",
        "testDescription": "y1",
        "steps": [
            "insert(person1)", "insert(person2)", "identify(person3, person1, person2)"
        ]
    },
    {
        "testId": "x2",
        "testDescription": "y2",
        "steps": [
            "insert(person1)", "insert(person2)", "identify(person3, person1, person2)"
        ]
    }
]
```
